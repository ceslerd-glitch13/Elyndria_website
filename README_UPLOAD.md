# Elyndria Creature Token Pack

This pack contains 120 original 256 × 256 WebP battle-map tokens. Each image is below 150 KB and uses the Elyndria site's 24-character content-derived asset ID.

## Repository-ready upload

Copy the contents of `docs/tokens/` into the repository's `docs/tokens/` folder, and replace `docs/token-index.json` with the included manifest. The current repository manifest was empty when this pack was prepared.

The `named/` folder contains byte-identical friendly copies for browsing. Do not upload those duplicates into `docs/tokens/`; the site only needs the coded files.

Use `catalog/token-catalog.csv` or `catalog/token-catalog.json` to match each creature name to its asset ID.

## Contents

- `docs/tokens/`: coded, repository-ready assets
- `docs/token-index.json`: website manifest
- `named/`: friendly filenames that include the code
- `catalog/`: name-to-code indexes and campaign metadata
- `SELECTION_NOTES.md`: coverage and exclusions
